# 📊 Ecommerce Analytics System

## 📌 Objetivo
Simular um sistema de e-commerce com análise de dados.

## ⚙️ Estrutura
- customers
- products
- orders

## 📊 O que o projeto faz
- Armazena clientes
- Registra pedidos
- Calcula faturamento
- Analisa comportamento de compra

## 🚀 Resultado
Projeto de análise de dados usando SQL com foco em business intelligence.
