# 🛒 Ecommerce Analytics System

Projeto de banco de dados e análise de dados de um sistema de e-commerce.

## 🔥 Funcionalidades
- Modelagem de banco de dados relacional
- Inserção de dados simulados
- Queries de análise (BI básico)
- Estrutura de projeto profissional

## 📊 Insights possíveis
- Faturamento total
- Clientes mais valiosos
- Volume de vendas

## 🧠 Tecnologias
- SQL (MySQL)
- Modelagem de dados
- Análise de dados

## 🚀 Status
Projeto funcional (executado via DB Fiddle)
