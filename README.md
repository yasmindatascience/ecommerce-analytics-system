# 🛒 Ecommerce Analytics System

## 📌 Sobre o Projeto
Este projeto simula um sistema de e-commerce com foco em modelagem de dados e análise de vendas utilizando SQL.

O objetivo é demonstrar como dados de clientes, produtos e pedidos podem ser estruturados e analisados para gerar insights de negócio.

---

## ⚙️ Tecnologias Utilizadas
- SQL (MySQL)
- Modelagem de banco de dados relacional
- Ambiente online de execução SQL

---

## 🧱 Estrutura do Banco de Dados
- customers → dados dos clientes
- products → catálogo de produtos
- orders → registros de compras

---

## 📊 Análises Implementadas
- Cálculo de faturamento total
- Identificação de clientes que mais gastam
- Quantidade total de clientes
- Ticket médio de pedidos

---

## 🚀 Como o Projeto Funciona

1. Criação das tabelas (schema SQL)
2. Inserção de dados simulados
3. Execução de queries analíticas
4. Geração de insights de negócio

---

## 💻 Execução do Projeto

O projeto foi executado utilizando um ambiente online de SQL:

👉 DB Fiddle (MySQL)

As queries foram testadas diretamente no ambiente e retornaram resultados válidos.
